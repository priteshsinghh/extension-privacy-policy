AI ORG Messenger Chrome Extension - Privacy Policy

This extension is designed for internal use within an organizational context. It captures web page HTML and URLs visited by users and transmits them to a secure backend server for AI processing.

Data Collected

  • Page URL and HTML content
  • Authenticated user ID (stored locally)
  • Authentication token (stored locally)

How We Use Data

The data is used for logging browsing behavior for productivity insights and AI-driven workflow automation.

Data Sharing

No data is shared with third parties. All collected data is stored securely on your organization's private infrastructure.

User Control

You may remove or disable the extension at any time. The extension does not track activity outside active usage.
